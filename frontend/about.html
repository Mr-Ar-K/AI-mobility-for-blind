About This Application

This application is a proof-of-concept for AI-powered object detection designed to assist visually impaired users.

It combines a powerful Python backend using FastAPI and YOLO models with a clear, accessible frontend to provide real-time audio feedback about objects detected in an uploaded video stream.

How It Works

  1. Upload: A user uploads a video file from their device.
  2. Process: The backend server receives the video, processes it frame by frame using advanced YOLO object detection models.
  3. Summarize: The system identifies all unique objects and their counts (e.g., "2 people", "1 car", "3 traffic lights").
  4. Speak: The summary is converted into a natural-sounding audio description using a Text-to-Speech (TTS) engine.
  5. Result: The user receives both the text summary and an audio file they can play to understand their surroundings.
